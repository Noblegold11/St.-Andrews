# 🏛️ St. Andrews Global | Elite Preparatory Portal

An elegant, high-performance academic landing page and admissions system designed for a premium educational institution. Built with a focus on "Quiet Luxury" aesthetics, editorial layouts, and seamless user experience.

## 🌟 Key Features

* **Premium Aesthetic:** Utilizes a custom Emerald & Amber color palette with refined typography (Playfair Display & Plus Jakarta Sans).
* **Dynamic Admissions Form:** Real-time email validation and simulated "Dossier Processing" with interactive loading states.
* **Faculty Registry:** A secure-style administrative dashboard that captures and displays inquiries using `localStorage`.
* **Editorial Campus Gallery:** A masonry-style grid with an immersive lightbox for high-resolution facility tours.
* **Responsive Architecture:** Fully optimized for mobile, tablet, and desktop using Tailwind CSS.

## 🛠️ Technology Stack

* **Frontend:** HTML5, Tailwind CSS (Utility-first styling)
* **Icons:** Lucide-Icons (Vector-based)
* **Typography:** Google Fonts API
* **Data Handling:** JavaScript `localStorage` (Client-side persistence)
